# IDLR — Internet Downloader

**Download Anything. Zero Limits.**

![Python](https://img.shields.io/badge/Python-3.10+-red)
![Flask](https://img.shields.io/badge/Flask-Web_App-black)
![yt-dlp](https://img.shields.io/badge/yt--dlp-Latest-red)
![License](https://img.shields.io/badge/License-MIT-green)

A beautiful, fast, local web app for downloading from YouTube, Spotify, TikTok, Instagram, SoundCloud and 1000+ other sites.

**Fast • Private • Unlimited • Self-Hosted**

---

## ✨ Features

| Feature             | Description                                        |
| ------------------- | -------------------------------------------------- |
| 🌐 1000+ Sites      | YouTube, Spotify, TikTok, Instagram, SoundCloud, X |
| 🎬 Video Downloads  | MP4, MKV, WEBM                                     |
| 🎵 Audio Downloads  | MP3, M4A, FLAC, OGG                                |
| ⚡ High Quality      | Up to 4K video and 320kbps audio                   |
| 📜 History          | Download history tracking                          |
| 👤 Accounts         | Local profiles and settings                        |
| 🌍 Network Sharing  | Access from any device on Wi-Fi                    |
| 🔄 One Click Update | Update yt-dlp directly                             |
| 🍪 Zero Tracking    | No cookies or analytics                            |

---

## 🚀 Quick Start

### Windows

```bash
pip install flask yt-dlp static-ffmpeg spotdl
python app.py
```

Open:

http://localhost:5000
