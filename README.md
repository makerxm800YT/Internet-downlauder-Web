# 🎥 YouTube Downloader

A clean, fast, and free YouTube (and 1000+ other sites) downloader built with Python + **yt-dlp**.

**Features:**
- Download single videos or playlists
- Video (MP4) or Audio only (MP3)
- Quality selection (up to 4K)
- Real-time progress bar + logs
- Choose download folder
- Simple & modern dark UI
- Auto-installs dependencies
